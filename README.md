# Exercícios de JavaScript

Este repositório contém uma coleção de exercícios de JavaScript desenvolvidos para praticar estruturas de controle, loops e funções. Cada exercício aborda um conceito específico de programação.

## Lista de Exercícios

1. **Verificar Par ou Ímpar**: Recebe um número inteiro e verifica se ele é par ou ímpar.
2. **Classificar Idade**: Classifica a idade de uma pessoa em categorias (criança, adolescente, adulto, idoso).
3. **Classificar Nota**: Recebe uma nota de 0 a 10 e classifica como "Aprovado", "Recuperação" ou "Reprovado".
4. **Menu Interativo**: Oferece um menu com três opções e executa a lógica correspondente à opção escolhida.
5. **Calcular IMC**: Calcula o Índice de Massa Corporal (IMC) e determina a categoria de peso.
6. **Verificar Triângulo**: Verifica se três lados fornecidos formam um triângulo e determina o tipo de triângulo.
7. **Calcular Preço das Maçãs**: Calcula o custo total de maçãs com base na quantidade comprada.
8. **Ordenar Valores**: Recebe dois valores e os exibe em ordem crescente.
9. **Contagem Regressiva**: Exibe uma contagem regressiva de 10 até 1.
10. **Escrever Número Várias Vezes**: Lê um número inteiro e o exibe 10 vezes.
11. **Somar Números**: Solicita ao usuário 5 números e calcula a soma total.
12. **Exibir Tabuada**: Exibe a tabuada de um número fornecido pelo usuário.
13. **Calcular Média**: Calcula a média aritmética de números inseridos pelo usuário até que o número 0 seja digitado.
14. **Calcular Fatorial**: Calcula o fatorial de um número fornecido pelo usuário.
15. **Sequência de Fibonacci**: Gera e imprime os primeiros 10 números da sequência de Fibonacci.

## Como Executar

1. Clone este repositório:
    ```bash
    git clone https://github.com/seu-usuario/seu-repositorio.git
    ```
2. Navegue até o diretório do projeto:
    ```bash
    cd seu-repositorio
    ```
3. Abra o arquivo `index.html` em um navegador ou utilize um ambiente de desenvolvimento que suporte JavaScript para executar os scripts diretamente no console.

## Requisitos

- Navegador web com suporte a JavaScript.
- Ambiente de desenvolvimento com console JavaScript (opcional).
